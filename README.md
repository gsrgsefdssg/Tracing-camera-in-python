# Face Tracking Camera

Программа для отслеживания лица и распознавания улыбок через веб-камеру.

## Возможности
- Обнаружение лица
- Трекинг лица в реальном времени
- Распознавание улыбки (Smiling/Neutral)
- Подсчет FPS
- Автоматическое восстановление при потере лица

## Установка
pip install opencv-python

## ⚠️ Примечание
Данный код написан нейросетью (ChatGPT) в образовательных целях.
Я использовал его, чтобы разобраться в работе OpenCV и трекинга лиц.
